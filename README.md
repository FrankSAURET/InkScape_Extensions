# InkScape Extensions Repository


## Français

Ce dépôt a pour unique objectif de stocker le fichier `list_of_extensions.json`, qui contient des informations sur chacune de mes extensions InkScape (nom, utilité, installation, liens, etc.).

Il n'y a pas d'autres fichiers ou fonctionnalités dans ce dépôt.

---


## English

This repository is solely intended to store the `list_of_extensions.json` file, which contains information about each of my InkScape extensions (name, purpose, installation, links, etc.).

There are no other files or features in this repository.
